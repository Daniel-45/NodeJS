Node.js es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor, basado en el lenguaje de programación JavaScript, asíncrono, utiliza un modelo de E/S de datos sin bloqueo, en una arquitectura orientada a eventos que lo hace muy ligero y eficiente frente a las aplicaciones en tiempo real de uso de datos y basado en el motor V8 de Google. Fue creado con el enfoque de ser útil en la creación de aplicaciones de red altamente escalables, como por ejemplo, servidores web.

https://nodejs.org/es/
